# pyPiTun
Easy the remote to the [pitunnel](https://www.pitunnel.com/).

#### Requirement :

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirement.

```bash
pip install -r requirement.txt
```

#### Usage :
##### **Linux**

```bash
ssh $(python3 pyPiTun.py)
```

##### **Windows**
```
in the process
```
